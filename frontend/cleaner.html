Sequence Cleaner

Trim, filter, and clean your FASTQ files with a comprehensive set of tools.

User Guide

How to Use

1. Upload File: Select the FASTQ file you want to process.

2. Configure Parameters: Adjust the cleaning settings below to fit your data.

3. Run: Click "Clean Sequence" to start the process.

4. Download: Once finished, a download button will appear for you to get the cleaned FASTQ file.

Parameter Guide
  • Adapter Sequence: The specific DNA sequence from the sequencing adapter to be removed from the 3' end of the reads.
  • Sliding Window Quality: Trims reads from the 3' end if the average quality within a moving window (typically 4-5 bp) drops below this Phred score.
  • Minimum/Maximum Length: Discards reads that are too short or too long after trimming.
  • Max Ambiguous Bases (%): Discards reads with a percentage of 'N' bases higher than this value.
