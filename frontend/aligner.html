Sequence Aligner

Align two DNA or protein sequences using the Needleman-Wunsch algorithm for optimal global alignment. This tool is designed for short sequences (e.g., genes, proteins) with a maximum length of 50,000 bp.

User Guide

How to Use

1. Input Sequences: Upload two FASTA files or paste the raw sequence text for Sequence 1 and Sequence 2.

2. Set Parameters: Adjust the scoring values to control how the algorithm penalizes mismatches and gaps.

3. Align: Click "Align Sequences" to see the results.

Parameter Guide
  • Match Score: The score given for identical characters. A higher value rewards matches more.
  • Mismatch Penalty: The penalty for non-identical characters. This should be a negative number.
  • Gap Penalty: The penalty for introducing a gap ('-'). A more negative number (e.g., -10 vs -1) makes gaps less likely, forcing the algorithm to prefer mismatches instead.
